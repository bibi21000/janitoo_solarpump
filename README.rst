==========
Rantanplan
==========

Another raspberry appliance server developped using Janitoo


Needs
=====

- PIR
- proximity
- temperature / humidity
- siren
- led status


Hardware
========

dht : temperature and humidity
gpio : PIR
1wire : temperature
...

Documentation
=============
You can browse online documentation here : https://bibi21000.github.io/janitoo_rantanplan/.
